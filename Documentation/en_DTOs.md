# Documentation - DTO's

## EmployeeDetailsDto

- Provides detailed information about an individual employee
- Employee data is displayed in the user interface.
  - Each employee has a dedicated profile page.

_Updated 10 Nov 2025_

## EmployeesSalaryDetailsDto

- Contains employee data relevant to payroll calculations.
  - For example: tax percentage and age.

**Usage**

- Salary Statements -page, Select one employee from the dropdown menu. Other form information is placed based on selected id (employee)
- Salary Statement calculations below the form

```C#

 public record EmployeesSalaryDetailsDto(
     Guid Id,                               // unique identifier
     string? FirstName,
     string? LastName,
     int? Age,                              // calculated from the employee’s date of birth
     decimal TyELPercent,                   // employment pension contribution rate (based on age)
     decimal UnemploymentInsurancePercent,  // unemployment insurance rate (based on age)
     decimal? TaxPercentage,                // tax rate provided by the Finnish Tax Administration
     decimal? SalaryAmount                  // gross salary before deductions
     );

```

_Updated Nov 10, 2025_

## EmployeesSummaryDto

- Provides a concise summary of employee information.
- All employees are listed in the application, showing only the most essential details.
  - List offers a link to view full details (information from EmployeeDetailsDto).

_Updated Nov 10, 2025_

## ErrorResponseDto

- Represents error messages returned from the backend.
- The Code field describes the type of error. For example, if no employees are found, it may contain a user-friendly message such as “No employees found.”
- The Message field can include a more detailed, developer-oriented description of the error generated by the system.

_Updated Nov 10, 2025_

## LoginRequestDto

- Represents the data sent to the backend when a user attempts to log in.
  - Username contains the value entered in the Username field of the UI.
  - Password contains the value entered in the Password field.

_Updated Nov 10, 2025_

## LoginResponseDto

- Represents the response returned from a login attempt.
- Success: indicates whether the login was successful.
- Message: an optional message, e.g., when login fails.
- UserRole: specifies the user’s role (if login was successful).
  - Examples: test user, admin, employee.

_Updated Nov 10, 2025_

## PayrollRatesDto

- Contains payroll rate information required for salary calculations.
  These rates are stored in the database and may change annually, so they must be maintained manually.
  - The employment pension (TyEL) rate (%) is verified from the Finnish Centre for Pensions (ETK).
  - The unemployment insurance rate (%) is verified from the Employment Fund (Työllisyysrahasto).

_Updated Nov 10, 2025_

## SalaryCalculationResultsDto

- Contains the results of salary-related calculations, expressed in euros.

_Updated Nov 10, 2025_

## SalaryStatementCalculationDto

- Represents a more comprehensive salary statement that includes both calculation results and employee information (e.g., name, tax rate, and gross salary).

```C#
    public record SalaryStatementCalculationDto(
        Guid EmployeeId,                        // 123
        string? EmployeeName,                   // Maija Mehiläinen
        decimal GrossSalary,                    // 5000 (€)
        decimal TaxPercent,                     // 22.0 (%)
        decimal WithholdingTax,                 // 1100.00 (€)
        decimal TyELAmount,                     // 357.50 (€)
        decimal UnemploymentInsuranceAmount,    // 29.50 (€)
        decimal NetSalary                       // 3513.00 (€)

        );
```
